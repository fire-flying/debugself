# debugself
![www.debugself.como](https://www.debugself.com)的blog源码

## 首次初始化 
先安装Node.js和git，然后执行如下命令
```
npm install
git clone https://github.com/gdbself/jacman.git themes/jacman
```
## 编写.md文件
markdown文件放到source/\_post目录

## 生成静态文件
```
npx hexo g
```

## 启动hexo的服务器
```
npx hexo server -p 8080
```
